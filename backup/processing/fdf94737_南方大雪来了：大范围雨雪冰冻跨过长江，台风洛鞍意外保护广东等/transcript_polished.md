# 南方大雪来了：大范围雨雪冰冻跨过长江，台风洛鞍意外保护广东等

大家好，我是中企爱。1月19日早晨起，今冬最强寒潮终于彻底集结完毕，开始大举南下。天气形势图可见，蒙古高原冷高压中心最高气压达到了1075百帕，为今冬以来最高。呼啸的北风迅速清扫西北、东北以及山西北部、京津冀北部的阴云，一头扎进黄河流域和长江流域的暖气流中。即使阳光普照，乌鲁木齐的最低气温也降到零下25度，北京的白天最高气温也只有零下7度，均为今冬以来最低。

而在山东、河南，寒潮刚刚下来时虽然势头够猛，地面温度降得够低，但在高空没有把暖湿气流全搞服帖，弄得冷一层暖一层，这就让河南、山东的水汽冻得不太得体，没有全部冻成雪花。正因为此，济南、郑州等地一开始是雪、冰粒、冻雨等各种东西齐下。很明显，这种天气非常坏，一方面让很多朋友看雪的心情大受影响，另一方面对交通运输造成了非常大的影响。很多朋友之前没有怎么见过冻雨，没把这个不起眼的小雨当回事。

河南几乎全境大雪，安徽北部也有中到大雪，江苏徐州出现暴雪。湖北北部、山东南部、陕西中部也下了雪。雨雪分界线迅速跨过淮河，已经推到了长江边。晚上8点前后，合肥已经下雪，武汉已经转为雨夹雪，南京也已经开始雨转雪。在雨雪之中，零度线已经推到了湖南北部、湖北东南部、安徽中部、江苏中部。

而接下来，随着寒潮的进一步南下，除四川盆地外，长江流域几乎全部转雪。其中湖北、安徽、江苏将在1月19日深夜到20日早晨完成转雪。1月20日早上，武汉、合肥、南京等地的早高峰很可能和雪碰上。因此，湖北、安徽、江苏以及湖南北部、江西北部、浙江西北部的朋友，1月20日早上如果一定要上班上学，务必早早起床，早早做出行准备，或者干脆就晚一点起床，错峰上学上班。

而当雪云抵达上海后，会迅速被干燥气流驱散，因此上海很可能就飘几个小时的雪，既可以欣赏到雪花，又不会造成积雪拥堵，可以放心地上班上学。

在寒潮大举南下之中，有三个被保护得很好的地方：一个是四川盆地，降温速度、幅度都比长江中下游要慢，成都、重庆市区也不会下雪，但最终仍然会非常阴冷，温度要逼近零度，重庆要降到5度左右；第二个是在云南，在乌蒙山的保护下，这次偏东的寒潮就真的放过云南了，除了昭通等地外，云南其他地方几乎不降温；第三个就是福建中南部、台湾、广东、海南以及广西东南部，这里虽然也降温，但降温幅度最终锁定在10度左右，几乎没有降水，是干冷型降温，体感上和长江中下游天差地别。实际上，直到1月19日下午，广东等地依然暖意融融，一派春天甚至初夏的气息。

为什么会这样？一方面当然是南岭、武夷山的功劳，但另一方面还要感谢一号台风“洛安”。正如我们之前节目所说，“洛安”台风虽然不强，但却是个体型非常庞大的台风，它的环流系统彻底屏蔽了广东、福建中南部、台湾、海南等地的水汽输送，让水汽在靠近岭南时迅速消散。可以说，一号台风“洛安”意外地保护了广东、福建、台湾、海南等地，让今冬最强寒潮最终对你们比较温柔。当然，温柔是相对的，寒潮仍然是强大的，岭南还是要冷一两天的，防寒保暖仍然是不可少的。
