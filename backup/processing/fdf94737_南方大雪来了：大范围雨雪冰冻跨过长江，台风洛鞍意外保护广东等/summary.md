# 南方大雪来了：大范围雨雪冰冻跨过长江，台风洛鞍意外保护广东等

## Summary
1月19日起最强寒潮南下，导致全国大范围降温降雪，影响交通出行，但部分地区受地形或台风影响降温较小。

### Key Facts
- 寒潮中心气压达1075百帕，为今冬最强
- 北方多地气温创今冬新低，如北京白天最高仅零下7度
- 河南、山东出现雨雪混合天气，影响交通
- 雨雪分界线迅速南推至长江边，多地转雪
- 四川盆地、云南、福建等地降温幅度较小或不受影响
- 台风洛安环流屏蔽水汽，意外保护广东、福建等地
