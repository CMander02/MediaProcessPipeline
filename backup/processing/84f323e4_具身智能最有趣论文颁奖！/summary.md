# 具身智能最有趣论文颁奖！

## Summary
清华大学助理教授华哲分享了2025年具身智能领域四类奖项的获奖论文，这些论文将科幻概念变为现实，涵盖动画角色实体化、简洁视觉提示、全身感知机器人及开源平台等方向。

### Key Facts
- 好莱坞Demo奖获奖论文《Olaf》将迪士尼动画角色带入现实世界
- 奥卡姆剃刀奖获奖论文《Aimbot》通过添加简单视觉准星提升机器人空间感知能力
- 分布外思考奖获奖论文《Robot Panoptes》设计全身配备21个摄像头的全能感知机器人
- 开放生态奖获奖论文《Beyond Mimic》提供易用的类人运动跟踪代码库降低研究门槛
- 其他提名包括乒乓球机器人《Hitter》、从演示学习相机运动的《Vision in Action》、揭秘生成式模型的《Much Ado About Noising》及开源平台《ToddlerBot》
