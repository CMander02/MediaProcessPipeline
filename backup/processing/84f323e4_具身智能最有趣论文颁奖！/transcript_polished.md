# 具身智能最有趣论文颁奖！

大家好，我是华哲，清华大学交叉信息研究院助理教授。我们上一期介绍了2025具身智能最值得读的十篇论文，那么今天给大家分享一些好玩的东西，就是2025年具身智能有哪些超酷的论文可以把科幻照进现实。

那么首先呢，是我们的第一个奖项，叫做好莱坞Demo奖，有两篇提名，它们都让科幻照进了现实。一篇叫做《Olaf: Bring an Animated Character to Life in the Physical World》，它把迪士尼的动画人物真的搬进了现实世界。另一个呢，叫做《Hitter: A Humanoid Table Tennis Robot via Hierarchical Planning and Learning》，让机器人学会打乒乓球。当然，最终当选的是《Olaf》。我想说，因为什么技术在可爱面前都不值一提。它不是好莱坞级别的demo，而是迪士尼级别的demo，因为它真的来自迪士尼。

然后是我们的奥卡姆剃刀奖，主打简洁高效，完美诠释了少即是多。第一篇提名的是叫做《Aimbot: A Simple Auxiliary Visual Cue to Enhance Spatial Awareness of Visual Motor Policies》。这个论文思路简直绝了，大家记得玩CS的时候在屏幕上用笔画准星的兄弟吗？那么这篇论文就是类似的思路，在机器人做任务的时候给它加一个类似的准星做视觉提示，居然它的能力就大幅提升了，超级简洁也超级好想。但是为什么想到的不是你？第二篇提名的是《Vision in Action: Learning Active Perception from Human Demonstration》，让机器人从人类演示里面学相机运动，不用复杂系统就能解决遮挡问题。那么最终当选的是《Aimbot》，因为《Aimbot》在简洁性上确实略胜一筹。

接下来是分布外思考奖，两篇都是在突破技术的边界。一篇我们在上一期已经介绍过了，叫做《Much Ado About Noising》，那么它确实揭秘了生成式模型在机器人控制中的成功秘密。而另一篇叫做《Robot Panoptes: The All-Seeing Robot with Whole Body Dexterity》，堪称全能感知机器人，全身装了21个摄像头，类似一条蛇，能够用身体的所有部位跟环境互动和感知。最后获奖的当然是让人过目难忘的《Robot Panoptes》。那么看到这样的机器人，谁不感叹一句：机器人还能这样？

最后是开放生态奖，两篇都是科研圈的福利担当。第一篇叫做《ToddlerBot: Open Source Machine Learning Compatible Humanoid Platform for Local Manipulation》，开源迷你类人机器人平台，续航强，兼容机器学习。不管你是学生、科研人员，还是在家的一个DIY爱好者，都能用上低成本的硬件做研究。第二篇叫做《Beyond Mimic: From Motion Tracking to Versatile Humanoid Control via Guided Diffusion》，提供了稳定易用的类人运动跟踪代码库，直接降低了类人控制的研究门槛，让多少科研人员少走弯路。最终《Beyond Mimic》胜出，毕竟那么多的机器人在跳舞，里面说不定藏着的都是《Beyond Mimic》的算法。
