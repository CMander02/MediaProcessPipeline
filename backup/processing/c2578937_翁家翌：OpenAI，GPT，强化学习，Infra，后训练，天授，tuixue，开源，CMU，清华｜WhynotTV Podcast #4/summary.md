# 翁家翌：OpenAI，GPT，强化学习，Infra，后训练，天授，tuixue，开源，CMU，清华｜WhynotTV Podcast #4

## Summary
嘉宾翁嘉义分享了他从童年对数学的兴趣、清华求学、开源项目、CMU硕士到加入OpenAI成为核心贡献者的经历，探讨了AI研究、工程能力、评价体系、AGI以及个人哲学思考。

### Key Facts
- 翁嘉义是OpenAI一系列模型（ChatGPT、GPT-4、GPT-4o、GPT-5）背后强化学习（RL）基础设施的核心贡献者。
- 他在清华期间开源了所有作业和资料，旨在打破信息差，认为“信息平权”很重要。
- 他开发了强化学习框架“天授”和签证查询系统“退学online”，视代码工具为一种慈善。
- 他认为在AI时代，工程能力和基础设施（Infra）的迭代速度比纯研究能力更重要。
- 他相信世界是确定性的（宿命论），但选择专注于当下体验和创造影响力。
- 他评价体系独特：看重GitHub star数、比赛成绩和论文，而非传统GPA或学历。
- 他认为OpenAI的成功关键在于人才密度、组织架构和信息流通效率。
- 他目前认为AI领域最需要的是Infra人才，以提升单位时间内的迭代次数和成功率。
